Tarot Adviser Tests (Command Prompt)

How to run:

1) Open Command Prompt in the repository root.
2) Run each test with Node:

   node doc\tarot-adviser-test\run-sample.js
   node doc\tarot-adviser-test\test-cache.js
   node doc\tarot-adviser-test\test-invalid-json.js

Optional Windows batch to run all:

   doc\tarot-adviser-test\run-all.cmd

These tests use a stub LLM client (no network access required).
